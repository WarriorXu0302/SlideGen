# PPT HTML Editor

一个强大的 HTML 演示文稿编辑器，支持从 AI 生成的 HTML 创建、编辑和导出精美的 PPT 文档。

## ✨ 主要功能

- 🤖 **AI 生成工作流**：通过优化的 Prompt 让大模型生成适配的 HTML
- 🎨 **可视化编辑**：双击元素直接编辑，所见即所得
- 📝 **源码编辑**：CodeMirror 代码编辑器，支持语法高亮
- 🔄 **撤销重做**：完整的历史记录，每个幻灯片独立管理
- 📱 **缩略图导航**：快速浏览和切换幻灯片
- 🎬 **全屏演示**：专业的演示模式，支持快捷键导航
- 📦 **多格式导出**：PPTX、PDF、PNG、JPEG 等格式

## 🚀 快速开始

### 安装

**Windows 用户：**
1. 下载 `PPT Editor-1.0.0-win.zip`
2. 解压到任意文件夹
3. 双击 `PPT Editor.exe` 启动


### 基本工作流程

1. **AI 生成 HTML** → 2. **导入编辑器** → 3. **微调优化** → 4. **导出 PPT**

## 🤖 AI 生成 HTML 指南

### 推荐 Prompt 模板

将以下 prompt 发送给 ChatGPT、Claude 或其他大模型：

```
你是一个专业的 PPT 设计师。请生成一个关于「[你的主题]」的 HTML PPT 文件。

严格遵守以下结构规范，每页用 <section data-slide="N" data-title="页面标题"> 包裹：

<!DOCTYPE html>
<html>
<head>
<style>
  body { margin: 0; padding: 0; }
  section[data-slide] {
    width: 1280px;
    height: 720px;
    position: relative;
    overflow: hidden;
    display: none;
  }
  /* 其他全局样式 */
</style>
</head>
<body>
  <section data-slide="1" data-title="封面">...</section>
  <section data-slide="2" data-title="...">...</section>
</body>
</html>

要求：
- 每页固定尺寸 1280×720px
- 使用内联 CSS，不引用任何外部资源（无外链字体、图片用 CSS 渐变/图标代替）
- 设计精美，有视觉层次感，配色统一
- 只输出完整 HTML 代码，不要任何解释文字

请生成 [N] 页关于「[你的主题]」的演示文稿。
```

### 示例用法

**生成商业计划书：**
```
你是一个专业的 PPT 设计师。请生成一个关于「AI 创业项目商业计划书」的 HTML PPT 文件。
[... 上述格式要求 ...]
请生成 8 页关于「AI 创业项目商业计划书」的演示文稿。
```

**生成技术分享：**
```
你是一个专业的 PPT 设计师。请生成一个关于「React 18 新特性详解」的 HTML PPT 文件。
[... 上述格式要求 ...]
请生成 12 页关于「React 18 新特性详解」的演示文稿。
```

### 关键约束说明

1. **固定尺寸**：每页必须是 1280×720px（16:9 比例）
2. **标准结构**：使用 `section[data-slide]` 包裹每一页内容
3. **内联资源**：所有 CSS 写在 `<style>` 标签内，图片用 CSS 渐变或 SVG 图标替代
4. **无外部依赖**：不能使用 Google Fonts、外部图片链接等
5. **语义化标题**：`data-title` 属性用于缩略图显示

## 📖 使用指南

### 导入 HTML

1. **创建新项目**：点击"打开文件"或直接拖拽 HTML 文件
2. **支持格式**：
   - `section[data-slide]` 格式（推荐）
   - `iframe[srcdoc]` 格式
   - 通用 `div.slide` 格式
   - 原始 HTML 文档

### 编辑功能

**可视化编辑：**
- 双击任意文本元素直接编辑
- Enter 键确认，Escape 键取消
- 修改会自动同步到源码

**源码编辑：**
- 点击"源码"按钮打开代码编辑器
- CodeMirror 语法高亮
- 自动保存修改

**撤销重做：**
- `Ctrl/Cmd + Z`：撤销
- `Ctrl/Cmd + Y` 或 `Ctrl/Cmd + Shift + Z`：重做
- 每个幻灯片独立记录历史

### 导航操作

- **缩略图**：点击左侧缩略图快速跳转
- **键盘导航**：`←→↑↓` 键切换幻灯片
- **演示模式**：`F5` 进入全屏演示

### 导出功能

1. **PPTX 格式**（推荐）：兼容 PowerPoint 和 WPS
2. **PDF 格式**：适合分享和打印
3. **图片格式**：PNG/JPEG，可选单张或打包

### HTML 生成建议
